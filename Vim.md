## vim
```bash
# 退出
:wq										# write and quit
:q!							 			# force quit

# 移动
w							 			# next word
b							 			# previous word
0							 			# beginning of the line
$							 			# end of the line
gg							 			# first line of the file
G							 			# last line of the file
Ctrl+f							 		# page down
Ctrl+b							 		# page up
50G							 			# line number 50

# 

```
